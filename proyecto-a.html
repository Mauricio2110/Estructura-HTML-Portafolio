Gestor de Tareas Inteligente

Un sistema que aprende de tu rutina para sugerir fechas y prioridades.

Objetivo

Ayudarte a organizar tus pendientes con recordatorios y recomendaciones basadas en tus hábitos de uso.

Tecnologías

  • Front-end: React
  • Back-end: Node.js + Express
  • Base de datos: MongoDB
  • Machine Learning: Python (scikit-learn)

Funcionalidades clave

  • Detección automática de tareas recurrentes.
  • Sugerencias de prioridad basado en historial.
  • Notificaciones push y email.
